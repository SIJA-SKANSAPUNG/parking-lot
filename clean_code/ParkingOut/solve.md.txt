- **Perbaikan:**
  - Tambahkan kedua kelas di folder `Utils`:
    - **`D:\21maret\clean_code\ParkingOut\Utils\Database.cs`:**
      ```csharp

      {
          public class Database
          {

              {

              }
          }
      }
      ```
    - **`D:\21maret\clean_code\ParkingOut\Utils\UserManager.cs`:**
      ```csharp
      namespace ParkingOut.Utils
      {

          {

          }
      }
      ```

      ```csharp
          {
      ```

      ```csharp
              public void LogError(string message, Exception ex = null) => Console.WriteLine($"{message} - {ex?.Message}");
          }

      ```

  - `"Argument 2: cannot convert from 'int' to 'System.Exception'"` (baris 175, 228)

### **Langkah Eksekusi**
1. **Tambahkan File ke `ParkingOut.csproj`:**
   ```xml
   <ItemGroup>
     <Compile Include="Utils\Database.cs" />
     <Compile Include="Utils\UserManager.cs" />
     <Compile Include="Utils\AppLogger.cs" />
     <Compile Include="Models\VehicleExit.cs" />
     <Compile Include="Services\VehicleEntryService.cs" />
     <Compile Include="Services\VehicleExitService.cs" />
     <Compile Include="Services\PrintService.cs" />
     <Compile Include="Services\CameraService.cs" />
     <Compile Include="UI\IconPaths.cs" />
   </ItemGroup>
   ```

2. **Build Ulang:**
   ```powershell
   cd D:\21maret\clean_code\ParkingOut
   Remove-Item -Path bin -Recurse -Force -ErrorAction SilentlyContinue
   Remove-Item -Path obj -Recurse -Force -ErrorAction SilentlyContinue
   dotnet restore
   dotnet build ParkingOut.csproj
   ```

### **Catatan**
- Jika masih ada error, kirimkan isi file seperti `App.xaml.cs`, `DashboardPage.xaml.cs`, atau `VehicleEntryPage.xaml.cs` untuk analisis lebih lanjut.
- Pastikan semua file di atas sudah dibuat dan namespace-nya sesuai.
